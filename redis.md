# Redis Server

## Install Redis

```shell
brew install redis
```

## Start Redis

```shell
brew services start redis
```

## Show Redis Info

```shell
brew services info redis
```

## Stop Redis

```shell
brew services stop redis
```

## Connect to Redis

```shell
redis-cli
```
